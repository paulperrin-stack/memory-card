# Memory Card 🎮

A Pokemon memory card game built with React. Test your memory by clicking each Pokemon only once - click the same one twice and it's game over!

## 🎮 Live Demo
[View it here](https://memory-card-nine-xi.vercel.app/)

## Features
- 5 progressively harder levels (6, 10, 12, 16, 20 cards)
- Cards shuffle on every click
- Tracks current score and best score
- Game Over and Levels Up overlays
- Pokemon fetched from the PokeAPI

## What I Learned
- Fetching data from an external API inside `useEffect`
- Using `Promise.all` to fetch multiple resources simultaneously
- Managing multiple pieces of state together
- Conditional rendering for game states
- Passing functions as props for child-to-parents communication
